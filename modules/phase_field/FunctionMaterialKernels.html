Function Material Kernels

Kernels CahnHilliard, CahnHilliardAniso, SplitCHParsed, and AllenCahn solve the Cahn-Hilliard and Allen-Cahn equations using free energies provided by Function Material objects. These include parsed function materials with free energy expressions supplied in the configuration files and all necessary derivatives to build the Residuals and Jacobian elements computed automatically using automatic differentiation (DerivativeParsedMaterial).

This enables rapid implementation of new Phase Field models without writing custom kernels and recompiling the code.
