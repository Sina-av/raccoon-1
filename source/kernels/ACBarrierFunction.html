ACBarrierFunction

The ACBarrierFunction kernel implements the term term for the free energy given in Moelans et al. (2008). is the barrier energy coefficient. In many phase field models, is constant, but in the case where it is a function of one or more of the variables, this kernel should be included.

Description

General Information

This kernel should only be used for variables that are used in calculating . Other variables will do a lot of math to return a value of 0.

and its derivatives should be calculated in a materials block that uses the DerivativeMaterialInterface, such as a DerivativeParsedMaterial.

The kernel only supports a single value of . It should not be necessary to use multiple 's in models with a non-constant .
