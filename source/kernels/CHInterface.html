CHInterface

Gradient energy Cahn-Hilliard Kernel with a scalar (isotropic) mobility

Implements the Cahn-Hilliard term for the gradient energy contribution for the isotropic mobility case. Its weak form is where (kappa_name) is the gradient energy coefficent, the conserved non-linear order parameter variable the kernel is acting on, (mob_name) is the scalar (isotropic) mobility associated with the order parameter, and is the test function.

See also

For an anisotropic version of this kernel see CHInterfaceAniso, which expects a tensor valued mobility .

    Options:

    Description:The kappa used with the kernel

    Options:

    Description:The mobility used with the kernel

    C++ Type:NonlinearVariableName

    Options:

    Description:The name of the variable that this residual object operates on
