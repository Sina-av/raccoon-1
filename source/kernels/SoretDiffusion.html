SoretDiffusion

Add Soret effect to Split formulation Cahn-Hilliard Kernel

Implements weak form

where is the concentration variable the kernel is operating on, the heat of transport, and the diffusivity.

It is used together with SplitCHWRes, SplitCHParsed, and CoupledTimeDerivative to set up a system of two first order PDEs using a concentration order parameter and a chemical potential variable.
