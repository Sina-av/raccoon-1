LangevinNoise

Source term for non-conserved Langevin noise

Adds a uniform noise source term with the weak form where is a random number in , (amplitude) is a constant amplitude factor, and (multiplier) is a material property that can be used as a location dependent mask for the source term.

warningwarning

This kernel does not produce parallel reproducible results. For a parallel reproducible random noise system see ConservedLangevinNoise

    C++ Type:double

    Options:

    Description:Amplitude

    Options:

    Description:The name of the variable that this residual object operates on
