Torque

Apply a moment as tractions distributed over a surface around a pivot point. This should operate on the displaced mesh for large deformations.

Overview

This boundary condition applies a torque (direction) with respect to a reference point (origin) through tractions distributed over a surface according to the right hand rule. It requires the calculation of the polar moment of inertia (or ), which can be performed using the PolarMomentOfInertia postprocessor.

The application of a torque is currently only validated for small strains.
