SwitchingFunction3PhaseMaterial

buildconstruction:Undocumented Class

The SwitchingFunction3PhaseMaterial has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

Material for switching function that prevents formation of a third phase at a two-phase interface: h_i = eta_i2/4 * [15 (1-eta_i) [1 + eta_i - (eta_k - eta_j)2] + eta_i * (9eta_i^2 - 5)]
