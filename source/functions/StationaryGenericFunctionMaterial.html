StationaryGenericFunctionMaterial

buildconstruction:Undocumented Class

The StationaryGenericFunctionMaterial has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

parses function material properties that are stationary in time. Computational cost is minimized by storing the stationary data but more memory is consumed in turn.
