Cohesive Zone Model Action

Description

The CohesiveZoneMaster action simplifies the input file syntax by adding the appropriate kernels required by a cohesive zone model. See CohesiveZoneMaster for more details.

Action to create an instance of the cohesive zone model kernel for each displacement component
